# Advanced Web Attack Techniques

Ce dépôt présente un ensemble de techniques avancées utilisées dans le cadre d’attaques ciblant les applications web.
L’objectif est **pédagogique** : aider à comprendre les vecteurs d’attaque modernes afin d’améliorer la sécurité des applications et d’adopter de bonnes pratiques défensives.

## Contenu

* **Bypass d’authentification** : exploitation de failles logiques, contournement de validations côté client, attaques sur les sessions.
* **Exploitation avancée d’injections** : SQLi modernes, NoSQLi, injections dans les parseurs ou ORM, attaques polyglottes.
* **Attaques côté client** : XSS avancés, UXSS, techniques de sandbox escape, pollution d’objets JavaScript.
* **Attaques sur l’infrastructure** : Server-Side Request Forgery (SSRF), template injection, attaques via des services cloud.
* **Techniques d’évasion** : obfuscation, payload mutables, contournement de WAF.

## Objectifs pédagogiques

* Comprendre comment les attaquants modernes procèdent.
* Apprendre à identifier les surfaces d’attaque.
* Renforcer la sécurité des applications via des contre-mesures adaptées.
